# Descrição do Projeto
O projeto foi desenvolvido com o objetivo de criar uma plataforma de venda de cursos online, visando aplicar conhecimentos adquiridos ao longo da formação acadêmica.
A plataforma foi estruturada para permitir cadastro de usuários, a disponibilização de cursos, a simulação do processo de compra e a organização dos conteúdos educacionais de maneira acessível e intuitiva.
